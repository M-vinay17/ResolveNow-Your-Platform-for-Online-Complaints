### 📽️ **Demo Highlights**

🔸 Register & Login
🔸 📄 Submit complaints with attachments
🔸 ⏱️ Track complaint status in real time
🔸 💬 Chat with assigned agents

🔸 🧠 Assign complaints to agents intelligently
